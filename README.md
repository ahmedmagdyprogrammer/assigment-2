Overview:

This project is a Simple Audio Player built using the JUCE framework in C++.
It allows the user to load, play, stop, and control multiple audio tracks simultaneously through an easy-to-use interface.

Features:

Play and stop audio tracks

Control volume (gain) and playback speed

Enable or disable looping

Two separate players that can play different audio files

Mixer to combine both audio outputs

Simple and clean GUI
